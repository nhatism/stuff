One click is all it takes! (Desktop use)

# Installation

0. Copy the code below.

1. Show the bookmarks bar if it is hidden.
   - **Chrome / Edge:** `Ctrl + Shift + B`

2. Right-click the bookmarks bar and select **Add page...**

3. Fill in:
   - **Name:** InsD or whatever you want
   - **URL:** Paste the bookmarklet code.

4. Save the bookmark.

![](/stuff/images/2026/insd.png)

---

# Usage

1. Open an Instagram post or story, or simply leave the desired photo visible on your screen while domscrolling your feed.
2. Click the **InsD** bookmark.
3. The original (hopefully) media will open in a new tab.
4. Save the file normally from your browser.

> Notes: Instagram Stories containing music, special effects, or certain interactive elements does not work. This bookmarklet only works with media currently visible on the screen. Instagram may change its website at any time, which could break this bookmarklet. But it's been working since 2021 so yeah =))
